This folder contains two subfolders (Subsets_22_06_84 and Subsets_31_12_84).

The "Subsets_22_06_84" subfolder containsthe following data sets and sample script:
1. Test site boundary in ESRI shapefile format (Hre_boundary.shp);
2. Landsat 5 Thematic Mapper (TM) data file "LT51700721984174XXX08_MTL.txt" and band files in tiff format, which was acquired on 22 June 1984 (dry season); and
3. Sample script for chapter 2: Pre-processing 

The "Subsets_31_12_84" subfolder containsthe following data sets:
1. Test site boundary in ESRI shapefile format (Hre_boundary.shp); and
2. Landsat 5 TM data file "LT51700721984366XXX02_MTL.txt" and band files in tiff format, which was acquired on 31 December 1984 (wet season).
